<item>Formatting</item	>
Formatting has a single attributes, <b>bold</b>, <r>reverse</r> and <u>underline</u>

and they look like this;


<pre>&lt;b&gt;BOLD&lt;/b&gt;
&lt;r&gt;Reverse&lt;/r&gt;
&lt;u&gt;underline&lt;/u&gt;</pre>


You can turn all elements off (for say, showing code) with a pre tag.

The above piece of code looks like this;

<pre>&lt;pre&gt;
&lt;b&gt;BOLD&lt;/b&gt;
&lt;r&gt;Reverse&lt;/r&gt;
&lt;u&gt;underline&lt;/u&gt;
&lt;/pre&gt;</pre>

Please note, mixing your attributes may not have the affect you
think it will, a Bold tag takes precendence over both Reverse and Underline,
and Reverse takes precendence over Underline. 


You can create a link to a subtopic (page title) like this

<item>SubTopic</item>

That code looks like this <pre>&lt;item&gt;SubTopic&lt;/item&gt;</pre>

The other feature is having the guide build a list of topics automatically
for you.

<list>SubTopic</list>

which looks like this <pre>&lt;list&gt;</pre>.

The LIST and LINK items both take up to 32 separators.

You can think of separators as subtitles. For example you can have text like;


<pre>
<item>Chapter 1</item>
text here...
<item>Chapter 1/Paragraph 1</item>
para 1 here
<item>Chapter 1/Paragraph 2</item>
para 2 here
<item>Chapter 1/Paragraph 3</item>
para 3 here
</pre>

and using a list as <pre><list>Chapter 1</list></pre> will show you a list like this

<pre>Paragraph 1
Paragraph 2
Paragraph 3
</pre>

And having a link jump directly to paragraph 3 you can use <pre> <link>Chapter 1/Paragraph 3</link> </pre>


<item>SubTopic 2</item>
This is a sub-topic.
