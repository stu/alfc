<item>Main/Introduction</item>

Welcome to Another Linux File Commander (<b>ALFC</b> for short).

This program is a file management utility patterned after
a very famous classic program called <b>Norton Commander</b>
(Hence the 'Commander' in the title as hommage).

This program is designed to be easy and powerful, giving you
control back over your filesystem.

<list>Main</list>


<item>Main/Date and Time Formats</item>

ALFC can display the time in the following formats;
<pre>
    HH:mm.ss - eg 13:57.03
    hh:mm.ss AMPM - eg 1:57.03 PM
    hh:mm.ssAMPM - eg 1:57.03PM
    hh:mm.ss - eg 1:57.03
    HH:mm - eg 13:57
    hh:mm - eg 1:57
    hh:mm AMPM - eg 1:57 PM
    hh:mmAMPM - eg 1:57PM
</pre>

    Where;

<pre>
    HH - 24hour format
    hh - 12hour format
    mm - minutes
    ss - seconds
    AMPM - Displays AM or PM
</pre>

Date formats are as below;
<pre>
    dd/mm/yyyy - eg 22/12/2009
    dd/mm/yy - eg 22/12/09
    mm/dd/yyyy - eg 12/22/2009
    mm/dd/yy - eg 12/22/09
    mm-dd-yyyy - eg 12-22-2009
    mm-dd-yy - eg 12-22-09
    dd-mm-yyyy - eg 22-12-2009
    dd-mm-yy - eg 22-12-09

    dd/mmm/yyyy - eg 22/dec/2009
    dd/mmm/yy - eg 22/dec/09
    mmm/dd/yyyy - eg dec/22/2009
    mmm/dd/yy - eg dec/22/09
    mmm-dd-yyyy - eg dec-22-2009
    mmm-dd-yy - eg dec-22-09
    dd-mmm-yyyy - eg 22-dec-2009
    dd-mmm-yy - eg 22-dec-09

    dd/mmmm/yyyy - eg 22/December/2009
    dd/mmmm/yy - eg 22/December/09
    mmmm/dd/yyyy - eg December/22/2009
    mmmm/dd/yy - eg December/22/09
    mmmm-dd-yyyy - eg December-22-2009
    mmmm-dd-yy - eg December-22-09
    dd-mmmm-yyyy - eg 22-December-2009
    dd-mmmm-yy - eg 22-December-09

    yymmdd - eg 091222
    yyyymmdd - eg 20091222
    ddmmyyyy - eg 22122009
    ddmmmyyyy - eg 22dec2009
</pre>

    Where;
<pre>
      dd - day of month (1-31)
      mm - month, numericly (1-12)
     mmm - month as jan/feb/.../dec
    mmmm - month as whole January, September, etc.
      yy - two digit year
    yyyy - four digit year
</pre>
